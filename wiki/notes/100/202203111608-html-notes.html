1.Common HTML Tags

HTML tag: <html>web page</html>

所有的網頁內容都在這個html tag之內

Header tag: <h1>header</h2>

<h2>subheadline</h2>還有h3，h4等。

Hyperlink tag: <a href="url">Link text</a>

url可以鏈接到文件、網站、文章等。Link text是url的description。

List tag:

Bulleted list - 無序列表

<ul>

<li>first list item</li>

<li>second list item</li>

<li>third list item</li>

</ul>

Number list 有序列表

<ol>

<ol>first list item</ol>

<ol>second list item</ol>

<ol>third list item</ol>

</ol>

Paragraph tag: <p>paragraph here</p>

文章段落

Image tag:

<img src="image file link" width="300" height="200"alt="keyword description">

添加圖片鏈接，可以設置圖片尺寸和添加圖片描述

Text formatting tags: 文字格式

Bold<strong>text here</strong> 粗體

Underline<u>text here</u> 下劃線

Italicize<em>text here</em> 斜體

字體的大小，字體樣式，尺寸，或者其他格式，可以使用css來實現

*Social sharing tags:*可能用不到

Facebook:

http://www.facebook.com/sharer/sharer.php?u=URL

添加將URL修改你需要的分享的URL，用戶點擊可直接跳轉過去

Title tag:<title>you title here</title>

爲當前頁面添加一個瀏覽器頂部標識

Body tag:<body>your webpage here</body>

Body標籤在html標籤裏面，Body標籤包含具體的網站內容，像list、paragraphs，hyperlinks，and images

Line break tag:</br>

將一行內容打斷成兩部分，break one line to two lines.可以在頁面的任何位置使用

Span tag: <span>item</span>

將一部分內容獨立出來，而不受整體格式的影響或者可以單獨出來給一個特定的格式
