如何优化自己的微信公众号

1、帐号定位

1.1 明确此公众号建立的目的

1.2 明确此公众号的服务对象

1.3 是否考虑商业变现（广告、软文等）

2、自定义菜单栏

2.1 与我联系之联系我

留下自己的个人微信号，让想与你取得联系的用户，能够方便的找到你。形式可以简约、美观：一个二维码；微信号；美化后的二维码等各种形式。

2.2 与我联系之关于我（非必要）

可以简单的介绍一下该公众号建立的目的与发展方向等等，让用户觉得此公众号有关注的必要。

2.3 其他菜单栏

自定义菜单栏不易过多，词条清晰明确，表达出此类别所包含的内容即可。类似文件夹的目录结构。 此子菜单内容可设置发送消息、跳转网页、跳转小程序。 跳转网页方面：可以将自己问文章进行整合，集中到某个大类下面，复制其链接，然后粘贴在这里。

3、话题

话题约等于标签。每篇文章的发表都会被打上标签，标签不易过多。单篇文章对应相关的标签。过多的标签不易归档整理。

3.1 公众号主页话题配置

如提示所说：可将话题标签配置在公众号主页上。方便用户一步到位去到该话题对应的文章集合。

3.2 页面模板

可新建一个公众号页面，可复制链接到自定义菜单发布。相当于文章集合。

4、被关注回复

当有新用户关注公众号后，会自动给用户发送一条讯息：文字、图片、音频、视频，自行选择。尽量有一些吸引力，给新用户一些好感。

5、图片水印

进入窗口：公众号设置->功能设置->图片水印

6、号内搜索

用户在公众号内进行搜索后，公众号推荐的话题。可方便用户快速定位到相关专题合集。

7、公众号介绍

给自己的公众号一个简练的介绍。 进入窗口：设置与开发->公众号设置->帐号详情。

8、问题

8.1 框架搭建

新开公众号一定要有自己的框架，而不是单纯的写文章发文章。对一些框架里暂时没有的内容可以，用建设中，敬请期待等字样，给用户一个明确的反馈，而不是404；网页不存在之类的。

8.2 编辑平台

请君自选。
