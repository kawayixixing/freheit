央视频下载步骤

原因: 普通的网页嗅探下载的央视频会出现花屏, 只有声音图像花屏. 于是在网络上寻找相关的的解决方法. 你别说还真有大佬给了解决方法, 在此进行整理.

1. 找视频

在网页端找到视频 ,在视频未播放前按F12进入浏览器的开发人员模式.

2. Network筛选

在network标签下的筛选器进行.do的链接筛选, 然后选择在新的标签页面打开.

3. 在这里了

在该页面Ctrl - F 检索hls_url 就是视频的地址了.然后拿到m3u8下载即可.

4. 使用ffmpeg下载

ffmpeg -i “m3u8的地址” 视频Name.mp4
