超级歪02-大脑学习法

1、学习与大脑的关系

1.1两种大脑思维模式

  • 专注模式

处于专注模式时，大脑负责处理高阶认知功能的前额叶会像控制中心一样，把你在思考的问题所牵动的不同部位串联起来。

处于专注模式时会有认知偏差。

定势效应 - 每个人都有一套不自觉已经习惯的思考方式，所以脑中最先浮现的念头反而可能会阻碍你找到更好的答案。

eg：西洋棋大师，虽然都会总览全局，但总是会专注棋盘上的同一位置。

  • 发散模式

把你的注意力从原本要解决的事情移开，去做一些不会让思绪紧绷的事情。当处于发散模式时，原本思考问题的视窗并不会被你的意识关掉，而是被放到意识的后台。这时候大脑中神经讯号的连接使四散的。

西洋棋13岁Carlsen，在最终决赛时，离开作为，让自己从专注转为发散。

需要主动在专注模式与发散模式中进行转换，前提是自己要先经历过专注模式，至少动脑思考过一次问题，才能把问题输入到脑子里，才可以促使发散模式以新的角度看事情。给大脑足够的时间预留给发散模式。

2、有效的学习方法

2.1 记忆与概念组块

组块 - 把一个个独立的咨询信息集合成一组有意义的概念。把接受到的这些信息连成一条线。

用各种方式去建构一个概念组块。eg：因果关系、时序关系、程度关系等。

理清你建立的这一组块概念是在那个脉络之内，也就是这个学科领域在处理什么问题的时候会用到这组概念。这样在遇到问题的时候就不用从一个个知识点开始回想，而是回想一整组概念。

把问题当成一首歌，在心里学着一遍一遍地唱，让信息合成一个流畅的组块，可以在任何时候调用。

2.2 回想能力错觉

读书划重点，会让人以为自己有划线就等于记来起来。

2.3 少量多餐的学习

想要学好某个领域的东西，你就必须给自己一点时间让大脑的神经连结有机会巩固。

eg：就像该一道砖墙，你应该是每一天铺点水泥，让它们有时间凝固，而不是在短时间内灌注大量水泥，这样建造的东西会歪不拉即。

一旦你理解来X，然后去读别的东西，改天再来复习X。

2.4 好好睡觉

人在清醒的时候，大脑机会产生有毒物质。人在睡觉的时候，人的脑细胞会收缩，细胞之间的空隙变大，可以让脑脊髓液清理掉白天积累的有毒物质，同时我们的大脑也会产生新的突触，去强化白天的学习记忆。

3、有效的记忆方法

3.1 三个方法增强记忆

  • 感官刺激阅读，记忆效果越强。

如果要增强记忆，试着把一个抽象的概念转换成一副难忘的画面。

  • 类比法

用隐喻的方法，去理解新的概念。电流-水流，用已知的解释未知的。

  • 间隔记忆

如果要把某个想法从短期记忆转变为长期记忆，需要符合两个条件：另人难忘；重复出现。

比较难记忆的东西，可以使用间隔式的复习方法。一次一次拉长复习的时间间隔。让大脑慢慢习惯你要建立的链接。

4、验收成功

如何检验自己懂了？

  • 看你能不能很流畅的转述给别人听。

我们常常是为了给某人说明一件事情，才会试着理解一个东西。而不是先理解来才说明。

  • 看你能不能举出相似的例子，或是自己能不能举个同样的例子
